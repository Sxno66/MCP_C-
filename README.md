  Le projet: Affichage de température avec un RaspBerry pi.
  En utilisant quelques outils nous vous proposons ce résultat.

**Le Matériels:**
Nous avons utiliser :

- Un Raspberry pi 3
- Un capteur de temperature ```MCP9808```
